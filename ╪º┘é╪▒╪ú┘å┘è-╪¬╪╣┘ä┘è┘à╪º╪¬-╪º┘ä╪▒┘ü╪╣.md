# موقع Noura Skin Secrets — النسخة الجديدة 🌸

## طريقة الرفع

ارفع **كل محتويات هذا المجلد** إلى جذر الاستضافة (نفس مكان الملفات القديمة)، مع استبدال الملفات الموجودة.

**مهم جداً — لا تحذف هذه الملفات الموجودة حالياً على السيرفر** (النسخة الجديدة تستخدمها ولم تتغير):

- `js/hudassistent.js` (شات مساعدة البشرة)
- `js/vendor-qrcode.js` (مولّد QR للسويش)
- مجلد `app/` كاملاً
- مجلد `images/` القديم (logo.jpg, om-oss.jpg, resultat-1.jpg, resultat-2.jpg)

يمكنك حذف: `css/style.css`, `css/noura-luxe.css`, `js/i18n.js`, `js/main.js`, `js/gallery.js`, `js/noura-luxe.js` (استُبدلت كلها بـ `css/site.css` والملفات الجديدة).

## بنية اللغات الجديدة

- السويدية: `nouraskinsecrets.se/` (الجذر)
- الإنجليزية: `nouraskinsecrets.se/en/`
- العربية: `nouraskinsecrets.se/ar/`

كل صفحة فيها روابط hreflang حتى يفهرس غوغل اللغات الثلاث بشكل صحيح.

## الصور — جاهزة ✓

صورك من GitHub صارت مربوطة بالكود. **ارفع مجلد `images/` كاملاً من الريبو إلى السيرفر** (نفس المسار `images/`). التوزيع:

- قبل/بعد: فيديو before-after.mp4 + صورة الكلف (publer-1770172237629) + صورة Before/After — والنسخة العربية فيها صورة "قبل/بعد" بالعربي
- معرض إنستغرام: IMG_2166 + publer-1757699814482 + publer-1768170262397 + publer-1771717340180 + resultat-3
- قسم "من نحن": IMG_2165 (صورة الجلسة)
- غلاف الهيرو: hero-poster.jpg

لتغيير أي صورة: عدّل قائمة GALLERY في `js/home.js` أو أسماء الصور في ملفات html.

## الفيديوهات والصور المطلوبة منك

الفيديوهات الحالية على السيرفر **تالفة** (حجمها ~100KB ولا تعمل). ارفع ملفات جديدة **بنفس الأسماء والمسارات**:

| الملف | المكان بالموقع | المواصفات المقترحة |
|---|---|---|
| `videos/hero.mp4` | خلفية الصورة الرئيسية | عمودي 4:5 أو مربع، H.264، بدون صوت، 10-20 ثانية، أقل من 8MB |
| `videos/before-after.mp4` | قسم قبل/بعد | H.264، أقل من 4MB |
| `images/hero-poster.jpg` | صورة الغلاف قبل تشغيل الفيديو | 1200×1500 تقريباً، أقل من 300KB |
| `images/om-oss.jpg` | قسم "من نحن" | صورتك أو صورة الصالون، 1200×1500 |
| `images/resultat-1.jpg` و `resultat-2.jpg` | قسم قبل/بعد | نتائج حقيقية 1200×1500 |
| `images/gallery/1.jpg` … | شريط إنستغرام | أي عدد، 800×1000 يكفي |

ملاحظات:

- إذا ملف ناقص أو تالف، **القسم يخفي نفسه تلقائياً** — الموقع لا ينكسر.
- لتفعيل صور المعرض: افتح `js/home.js` وأضف أسماء الملفات في قائمة `GALLERY` (الشرح موجود داخل الملف).
- لضغط الفيديو: صدّر من الهاتف بجودة 1080p ثم استخدم موقع مثل handbrake أو أي ضاغط MP4.

## المراجعات

افتح `js/reviews.js` وألصق 3-6 مراجعات حقيقية من غوغل/Bokadirekt بهذا الشكل:

```js
const REVIEWS = [
  { stars: 5, text: "نص المراجعة...", name: "Sara", source: "Google" },
];
```

قسم "ماذا تقول عميلاتنا" يظهر تلقائياً أول ما تضيف أول مراجعة (حالياً مخفي).

## البريد الإلكتروني

الموقع حالياً يستخدم info@nouraskinsecrets.se. عندما تنشئ بريداً على الدومين (مثل info@nouraskinsecrets.se) من لوحة الاستضافة، ابحث واستبدل العنوان في كل ملفات html + `js/booking.js` + `js/payment.js`، وحدّث بريد الاستقبال في حساب Formspree.

## تعديل المحتوى لاحقاً

- الأسعار والعلاجات: `js/treatments.js` (كل علاج له نص سويدي/إنجليزي/عربي)
- العروض: `js/offers.js` — الباقات: `js/klippkort.js`
- النصوص الثابتة للصفحات: عدّل `build.py` ثم شغّل `python3 build.py` ليعيد توليد الـ12 صفحة، أو عدّل ملفات HTML مباشرة.
