# Noura Skin Secrets - تطبيق الهاتف الذكي

تطبيق متعدد المنصات (iOS و Android) لخدمات Noura Skin Secrets، مبني باستخدام React Native و Expo.

## المميزات الرئيسية

### 1. عرض الخدمات والعروض 📋
- عرض قائمة شاملة بالخدمات المتاحة
- عرض التفاصيل والأسعار والمدة الزمنية
- عرض الخصومات والعروض الخاصة
- البحث والتصفية

### 2. نظام الحجز 📅
- اختيار الخدمة المطلوبة
- اختيار التاريخ والوقت
- إضافة ملاحظات خاصة
- عرض الأوقات المتاحة
- تأكيد الحجز الفوري

### 3. ملف العميل 👤
- عرض وتعديل البيانات الشخصية
- إضافة معلومات البشرة والحساسيات
- عرض صورة الملف الشخصي
- الإحصائيات الشخصية (عدد المواعيد، المبلغ المنفق)

### 4. إدارة المواعيد 📝
- عرض جميع المواعيد المؤكدة والمعلقة والمكتملة
- تعديل المواعيد المؤكدة
- إلغاء المواعيد
- تصفية المواعيد حسب الحالة

### 5. نظام التقييمات والتعليقات ⭐
- تقييم الخدمات بعد الانتهاء منها
- إضافة تعليقات وملاحظات
- عرض تقييمات المستخدمين الآخرين
- التحقق من التقييمات الموثوقة

### 6. الدفع الإلكتروني 💳
- دعم بطاقات الائتمان
- دعم Apple Pay و Google Pay
- عرض طرق الدفع المحفوظة
- إدارة آمنة للدفع

## البدء السريع

### المتطلبات
- Node.js 16+ و npm أو yarn
- Expo CLI
- أجهزة محاكاة iOS و Android (اختياري)

### التثبيت
```bash
# تثبيت الحزم
npm install

# تشغيل التطبيق
npm start

# للـ iOS
npm run ios

# للـ Android
npm run android
```

### متغيرات البيئة
1. انسخ `.env.example` إلى `.env`
2. أضف الأمفتاح والبيانات التالية:
```
EXPO_PUBLIC_API_URL=https://your-api-url.com
EXPO_PUBLIC_STRIPE_KEY=your_stripe_key
```

## التكامل مع الخادم

### API Endpoints المطلوبة

#### الخدمات
- `GET /services` - الحصول على جميع الخدمات
- `GET /services/:id` - الحصول على تفاصيل الخدمة
- `GET /services/search` - البحث عن الخدمات

#### المواعيد
- `GET /appointments` - الحصول على المواعيد
- `POST /appointments` - إنشاء موعد جديد
- `PATCH /appointments/:id` - تحديث موعد
- `DELETE /appointments/:id` - إلغاء موعد

#### التقييمات
- `GET /services/:id/reviews` - الحصول على تقييمات الخدمة
- `POST /reviews` - إضافة تقييم جديد

#### ملف العميل
- `GET /client/profile` - الحصول على بيانات العميل
- `PATCH /client/profile` - تحديث بيانات العميل
- `POST /client/profile-picture` - تحديث صورة الملف

#### الدفع
- `POST /payments/intent` - إنشاء بيانات دفع
- `POST /payments/:id/confirm` - تأكيد الدفع
