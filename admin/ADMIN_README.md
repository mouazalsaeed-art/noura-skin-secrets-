# لوحة تحكم Noura Skin Secrets Admin

تطبيق React احترافي لإدارة نظام Noura Skin Secrets.

## المميزات

✨ **لوحة بيانات متقدمة**
- رسوم بيانية تفاعلية
- إحصائيات شاملة
- مؤشرات الأداء الرئيسية (KPIs)

📦 **إدارة الخدمات**
- إنشاء وتحرير وحذف الخدمات
- إدارة الأسعار والخصومات
- تصنيف الخدمات

📅 **إدارة المواعيد**
- عرض جميع المواعيد
- تأكيد وإكمال المواعيد
- تصفية حسب الحالة
- عرض ملاحظات العملاء

⭐ **إدارة التقييمات**
- مراجعة التقييمات المعلقة
- التحقق من التقييمات
- حذف التقييمات غير المناسبة

👥 **إدارة المستخدمين**
- عرض قائمة المستخدمين
- عرض تفاصيل الملف الشخصي
- حذف الحسابات

🔐 **نظام أمان شامل**
- مصادقة JWT
- تحكم بالوصول
- حماية المسارات

## المتطلبات

- Node.js 16+
- npm أو yarn
- خادم API يعمل على `http://localhost:5000`

## التثبيت

### 1. تثبيت الحزم
```bash
cd admin
npm install
```

### 2. إعداد متغيرات البيئة
```bash
cp .env.example .env
# عدّل VITE_API_URL إذا لزم الأمر
```

### 3. بدء خادم التطوير
```bash
npm run dev
```

سيفتح التطبيق تلقائياً على `http://localhost:3000`

## نصائح الاستخدام

### تسجيل الدخول
1. افتح التطبيق على `http://localhost:3000`
2. أدخل البيانات المتوفرة أعلاه
3. انقر على "دخول"

### إضافة خدمة
1. اذهب إلى صفحة "الخدمات"
2. انقر على "إضافة خدمة"
3. ملء البيانات
4. انقر على "إضافة"

### تأكيد موعد
1. اذهب إلى "المواعيد"
2. انقر على ✓ للتأكيد
3. المرة التالية انقر ✓ للإكمال

### التحقق من التقييم
1. اذهب إلى "التقييمات"
2. حدد "غير مُتحقق"
3. اقرأ التقييم
4. انقر على "التحقق"
